Este proyecto fue creado con una arquitectura SPA, es decir Back y Front independientes, y utilizando un BD MySQL relacional
a continuación dare las especificaciones básicas para levantar el/los proyectos:


Backend: Creado en Java con SpringBoot y con el gestor de paquetes Maven, 
		requiere java 17, ejecutar la aplicación con el botón 'RunApplication', IDE libre, solo que sea compatible 
		con proyectos java.

IDE utilizado por el aspirante: IntelliJ IDEA 2024.1.2


FrontEnd: Creado en angular Framework, requiere tener instalado lo siguiente -> 
		Node v 12.22,
                Angular Cli : 11.2.19
                En una terminal en el IDE de preferencia ubicarse en la carpeta del proyecto 'abcc-articulos'
                e ingresar el siguiente comando 'npm install', al finalizar su ejecución, levantar el proyecto
                con el siguiente comando 'ng serve'
IDE utilizado por el aspirante: WebStorm 2023.3.2

Base de datos: Importar la base de datos sql en cualquier software gestor de base de datos y hacer una conexión local (Tener habilitado MySQL de forma local por ejemplo con ayuda de software como: Wamp64 o Xampp )
                 (Conexión con el back mediante jdbc en el application.properties).

Software Utilizado por el apirante: DBeaver 24.1.2
